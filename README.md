# Happy Snake \uD83D\uDC0D

A family-friendly, cheerful Snake game built with Jetpack Compose.

## Features
- **Happy Visuals**: Bright colors and a friendly-looking snake with eyes!
- **Simple Controls**: Easy-to-use directional buttons.
- **Score Tracking**: Keep track of how many apples your snake eats.
- **Family Friendly**: Positive messaging and "Game Over" screens.

## Technical Details
- Built entirely using **Jetpack Compose**.
- Uses Material 3 components.
- Smooth game loop using Kotlin Coroutines (`LaunchedEffect`).

## How to Play
1. Launch the app.
2. Use the arrow buttons to move the snake.
3. Eat the red apples (\uD83C\uDF4E) to grow and increase your score.
4. Don't run into your own tail!
5. Have fun! \uD83D\uDE0A
